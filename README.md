# Bodø Bokseklubb

React-nettsiden kjører med Vite.

## Kom i gang

Installer avhengighetene og start utviklingsserveren:

```sh
npm install
npm run dev
```

Vite viser den lokale adressen i terminalen, vanligvis `http://localhost:5173`.

## Kommandoer

- `npm run dev` starter utviklingsserveren med hot reload.
- `npm run build` lager en produksjonsbygning i `dist/`.
- `npm run preview` forhåndsviser produksjonsbygningen lokalt.
